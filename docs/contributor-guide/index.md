## Overview

OSPA is an open-source project, and we welcome contributions of all kinds:

- Bug reports and feature requests
- Documentation improvements
- Code contributions
- Testing and feedback

## Ways to Contribute

### For Everyone

- **Report bugs** - Found a problem? [Open an issue](https://github.com/OpenStack-Policy-Agent/OSPA/issues)
- **Suggest features** - Have an idea? We'd love to hear it
- **Improve documentation** - Fix typos, add examples, clarify content
- **Share feedback** - Tell us about your experience using OSPA

### For Developers

- **Fix bugs** - Check the [issue tracker](https://github.com/OpenStack-Policy-Agent/OSPA/issues) for open bugs
- **Add features** - Implement new functionality
- **Add services** - Extend OSPA with new OpenStack service support
- **Write tests** - Improve test coverage
- **Review PRs** - Help review pull requests from others

## Getting Started

1. Read the [Contributing Guide](contributing.md)
2. Review the [Code of Conduct](code-of-conduct.md)
3. Set up your [development environment](../developer-guide/setup.md)
4. Find an issue or feature to work on
5. Submit a pull request

## Community

- **GitHub Discussions** - Ask questions and share ideas
- **Issue Tracker** - Report bugs and track features
- **Pull Requests** - Submit and review contributions

## Recognition

All contributors are recognized in:

- GitHub contributors list
- Release notes for significant contributions
- CONTRIBUTORS file in the repository
