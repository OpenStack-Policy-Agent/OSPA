# User Guide

This guide covers everything you need to know about using OSPA to audit your OpenStack cloud.

## Overview

OSPA operates in two modes:

1. **Audit Mode** (default) - Scans resources and reports violations without making changes
2. **Remediation Mode** - Applies fixes when `--fix` is enabled
